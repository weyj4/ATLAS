### Build Instructions

These instructions assume that you have already setup the PostgreSQL database

1. Clone this repository `git clone https://github.com/ArnholdInstitute/ATLAS.git`
2. `cd ATLAS`
3. `npm install`
4. `npm run dev`
5. Open up `localhost:8080` in your web browser
